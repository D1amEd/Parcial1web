## Correr react
1. Descargar el zip o clonar el proyecto de github
2. Entrar al archivo y correr npm i
```sh
cd Parcial2Web
npm i
```
3. Una vez se entre, correr react con 
```sh
npm start
```
El front debería correr en [http://localhost:3000](http://localhost:3000)
